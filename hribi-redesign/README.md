# HribiRedesign: Navodila za namestitev in zagon

### 1. Namestitev dependencijev
V mapi `hribi-redesign` zaženite:
```bash
npm install
```

### 2. Nastavitev okoljskih spremenljivk
Kopirajte predlogo in dodajte svoj API ključ:
```bash
cp src/environments/environment.template.ts src/environments/environment.ts
```

### 3. Zagon aplikacije
```bash
ng serve --open
```
Aplikacija bo dostopna na `http://localhost:4200/`.

### 4. Uporaba aplikacije
V aplikaciji na zemljevidu ali v iskalniku poiščite Triglav, Goloved ali Snežnik za dostop do podrobnejših informacij o vrhu. Dodatne funkcionalnosti, kot so vreme, novice in zemljevid s filtriranjem so na voljo v meniju (izbirni vrstici na vrhu)
